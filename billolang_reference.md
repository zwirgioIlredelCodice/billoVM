## operations

    * `+`
    * `-`
    * `*`
    * `/`
* logic
    * `and`
    * `or`
    * `xor`
    * `not`

## keywords
* control flow
    * `if`
    * `while`
    * `do`
    * `end`

## use
* **operations**
    * `+`, `-`, `*`,`/`
    ```
    + x y z ->
    - x y z ->
    ....
    ```

* **control flow**
    * `if` basic
        ```
        if -> condition -> do ->
            code ->
        end ->
        ```
    * `else` basic
        ```
        if -> condition -> do ->
            code ->
        else ->
            code ->
        end ->
        ```
    * `while` basic
        ```
        while -> condition -> do ->
            code ->
        end ->
        ```  
* **system call**
    * `output` basic
        ```
        65 -> output ->
        ```
